La distillation d'alcool a domicile est INTERDITE dans de nombreux pays sans licence.

Ce guide est fourni a titre educatif uniquement.

Risques : explosion des vapeurs, brulures thermiques, responsabilite penale.

Methanol : Le vin de raisin contient tres peu de methanol (40-120 mg/L). La distillation de vin blanc ne presente PAS de risque significatif de methanol. Le methanol se concentre dans les queues (pas les tetes). L'elimination des tetes sert a retirer l'acetone et l'acetaldehyde (qualite gustative).

Principe fondamental

La distillation SEPARE l'ethanol de l'eau. Elle ne cree jamais d'alcool. On part de 0.240L d'ethanol pur et on ne peut en recuperer qu'une fraction (pertes a chaque etape).

Acces rapide

Concentrer rapidement l'ethanol du vin en un "bas-vin" (low wines). Pas de coupes tetes/coeur/queues. Eliminer seulement les foreshots. Chauffer vite.

Bilan D1

Entree : 2 000 mL de vin a 12% = 240 mL ethanol pur

Foreshots elimines : 10-15 mL (traces d'ethanol perdues)

Residus dans le pot : ~5-10% de l'ethanol (~15-24 mL)

Evaporation/adherence : ~2-3%

Sortie : ~600 mL de bas-vin a ~35% ABV = ~210 mL ethanol pur

Recuperation : 210/240 = ~88%

"Regle du tiers" : volume bas-vin ~ 1/3 du volume de charge. 2000/3 = 667 mL. Coherent avec ~600 mL.

Methode d'infusion

Maceration directe (12-24h) :

  1. Ajouter les botaniques dans le bas-vin, couvrir
  2. Macerer 12-24 heures a temperature ambiante (20°C)
  3. Ne pas depasser 48h (amertume excessive)
  4. Charger le tout dans l'alambic et distiller lentement

Panier a vapeur (gin basket) si disponible :

  • Botaniques robustes (genievre, coriandre, angelique) : dans le pot, maceration 12-24h
  • Botaniques delicats (agrumes, cardamome) : dans le panier a vapeur
  • Les vapeurs traversent le panier pendant la distillation
  • Resultat plus fin et elegant (methode des gins premium)

Identification des fractions D2

FORESHOTS - A ELIMINER DEFINITIVEMENT

Volume : 5-10 mL (~5 mL/L de charge)

  • Odeur : piquante, solvant, chimique (acetone)
  • JAMAIS consommer ni recycler

TETES - A COLLECTER SEPAREMENT

Volume : ~15-25 mL

  • Odeur : moins agressive que foreshots, encore chimique
  • Recyclable : ajouter aux prochaines distillations

COEUR - LE GIN (A COLLECTER)

Volume attendu : ~150-200 mL a 70-75% ABV

  • Debut : odeur propre, cristalline, notes de genievre, ~78% ABV
  • Milieu : equilibre parfait, aromes botaniques clairs
  • Fin : surveiller apparition d'odeur lourde/grasse/carton mouille
  • Tests : frotter entre les paumes et sentir, ou diluer a 40% et gouter
  • Apparence : parfaitement limpide, incolore

QUEUES - A COLLECTER SEPAREMENT

Volume : ~50-100 mL

  • Odeur : carton mouille, savon, graisse
  • Contient le plus de methanol et d'alcools de fusel
  • Recyclable dans une future D1 (ou a eliminer)

Verification : 15 + 128 + 28 + 39 = 210 mL ethanol. Bilan equilibre.

Dilution a 40% ABV et Finition

Formule de dilution

Volume d'eau = V_esprit × (ABV_initial / ABV_final - 1)

Approximation. Contraction de volume ~2-3%. Toujours verifier a l'alcoometre apres repos.

Procedure de dilution

Etape 1 : Preparation

  • Eau : osmose inverse (RO) ideale. Eau distillee acceptable. TDS < 10 ppm.
  • L'eau de source peut causer du trouble (calcium/magnesium).
  • Eau et distillat a la MEME temperature (~20°C).

Etape 2 : Melange

  1. Verser l'eau DANS l'esprit (jamais l'inverse) - empeche le louche
  2. Verser en filet fin, en remuant doucement
  3. Idealement, diluer par etapes : d'abord a 60%, repos 2 jours, puis a 40%
  4. Mesurer l'ABV a l'alcoometre apres chaque ajout

Etape 3 : Repos / Mariage (CRITIQUE)

Duree : 2 a 3 semaines minimum (standard professionnel)

Minimum absolu : 7 jours

Optimal : 4-6 semaines

Pourquoi si long ?

  • Reorganisation des liaisons hydrogene eau-ethanol
  • Stabilisation des terpenes du genievre
  • Equilibration des esters
  • Integration des huiles essentielles botaniques

Etape 4 : Verification finale

  • Alcoometre : 40% +/- 1%. Si > 42% : ajouter eau. Si < 38% : ajouter coeur pur.
  • Limpidite : parfaitement transparent. Si trouble : filtrer ou laisser reposer.
  • Odeur : genievre dominant, agrumes, epices. Pas d'odeur chimique.
  • Gout (5 mL dilue) : equilibre, pas agressif.

Exemple complet : lot type

Coeur D3 obtenu : 135 mL a 80% ABV

Ethanol pur : 135 x 0.80 = 108 mL (parti de 240 mL, pertes 55% : normal)

Eau a ajouter : 135 x (80/40 - 1) = 135 mL d'eau RO

Volume final : ~270 mL (contraction ~2% => ~264 mL reel)

Repos : 3 semaines en bocal verre ferme, 18°C, obscurite

Resultat : ~265 mL de gin a 40% ABV
